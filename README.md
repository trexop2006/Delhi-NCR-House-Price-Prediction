# 🏠 Delhi/NCR House Price Prediction

A Machine Learning web application that predicts the estimated price of residential properties in Delhi/NCR using property details such as area, bedrooms, bathrooms, furnishing status, building type, and city.

The application is built using **Python, Scikit-learn, Random Forest Regressor, and Streamlit**.

---

## 🚀 Features

- Predict house prices in Delhi/NCR
- Interactive Streamlit web interface
- Random Forest Regressor model
- Automatic Latitude & Longitude mapping based on selected city
- One-Hot Encoding for categorical features
- Input validation
- Instant house price prediction

---

## 🧠 Machine Learning Workflow

- Data Collection
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Encoding Categorical Features
- Train-Test Split
- Model Training
- Model Evaluation
- Model Serialization using Joblib
- Streamlit Web App Deployment

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/trexop2006/Delhi-NCR-House-Price-Prediction.git
```

Move into the project folder

```bash
cd Delhi-NCR-House-Price-Prediction
```

Install the required dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```
